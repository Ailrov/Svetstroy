Цоколи чугунные для опор освещения

Цоколь представляет собой сборно-литую конструкцию, выполненную из чугуна. Для доступа к опоре имеются технологические отверстия, закрываемые лючками.

Цоколь чугунный Ц-5 предназначен для стальных силовых опор ОС 0,4-9,0.

Диаметр верхний внутренний для опоры 219 мм.

Диаметр внешний верхний 276 мм.

Высота цоколя 1500 мм.

Нижний диаметр 590 мм.

Размеры лючка 420х240 мм.

Диаметр отверстий крепления 22 мм.
